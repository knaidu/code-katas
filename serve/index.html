About

This book will help you get started with programming data structures and algorithms using Ruby programming language. You will be able to analyze performance of different data structures and algorithms. The book covers most of the core concepts of typical undergraduate level data structure and algorithm courses.

The book places strong emphasis on actually implementing and running data structures and algorithms. For that reason, all the code in the book are real Ruby codes that can readily be implemented.

The book is still under progress. The content of book will be updated frequently.
